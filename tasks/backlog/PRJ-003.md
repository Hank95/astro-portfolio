# Add GitHub Actions CI pipeline

## Objective
No CI/CD exists. A basic GitHub Actions workflow would catch build failures, type errors, and broken links before they hit production on Cloudflare Pages.

## Acceptance Criteria
- [ ] `.github/workflows/ci.yml` runs on push and PR
- [ ] Pipeline runs `astro check` (TypeScript validation)
- [ ] Pipeline runs `astro build` to verify the site builds
- [ ] Pipeline status is visible on PRs

## Notes
