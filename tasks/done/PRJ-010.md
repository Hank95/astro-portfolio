# Auto-generate OG images per blog post

## Objective
Generate unique Open Graph social card images for each blog post at build time. Currently every page shares the same generic OG image, which looks bland when posts are shared on LinkedIn, Twitter, etc.

## Acceptance Criteria
- [ ] Each blog post gets a unique OG image with the post title and site branding
- [ ] OG images are generated at build time (not runtime)
- [ ] Images render correctly in social media preview tools
- [ ] Fallback to the generic OG image for non-blog pages

## Notes
Options: `satori` + `sharp` (most common Astro approach), `@vercel/og`, or a custom Canvas-based solution. The image should match the site's typography and color palette.
