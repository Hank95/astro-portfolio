# Add project hero images to major projects

## Objective
Only Strava Local has screenshot/image assets. The other featured projects (Klew, Apex146, F1 Dashboard, ListLive) have no visual assets, making the project detail pages and cards less compelling. Add hero images or screenshots to at least the 5 featured projects.

## Acceptance Criteria
- [ ] Each featured project has at least one hero image defined in `projects.ts`
- [ ] Images are optimized (compressed, appropriate dimensions)
- [ ] Project detail pages render images correctly
- [ ] Project cards on homepage show images where applicable

## Notes
Current image support: `image?: string` and `images?: { src, alt, caption }[]` fields exist on the Project interface.
Strava Local images live in `public/strava-local/`.
